# Needit
Spring MVC 프로젝트

결제 시스템을 갖춘 지역 기반 물품, 인력, 공간 대여 사이트

# 프로젝트 정보

## 1. 프로젝트 기간
* 2023.10.10 ~ 2023.12.05

## 2. 개발 인원
* 장기흥: 게시판, 예약, 결제

* 이중배: 회원가입, 소셜로그인

* 김종민: 게시판, 마이페이지

* 남경민: 메인페이지, 관리자페이지

* 박근형: 채팅, 지도

* 이주현: 게시판, 예약, 찜
